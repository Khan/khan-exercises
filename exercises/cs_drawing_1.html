A point consists of a single pair of X and Y coordinates.

A point is usually a single pixel, drawn using the syntax: 
point(x, y);

Remember: In the arguments to point (and all other drawing functions) the X coordinate comes before the Y coordinate.

To draw your point you would need to write: 
point(x1, y1);

However, that's not enough: You need to adjust the strokeWeight of the point as well!

Remember: If you want to change the strokeWeight it needs to occur before the shape you're drawing.

Your complete code will probably look something like this: 
strokeWeight(10);
point(x1, y1);

A line is drawn using two points, one for the start of the line and one for the end of the line.

A point consists of an X and a Y coordinate. Thus a line will take two pairs of X and Y coordinates.

The syntax for a line is as follows: 
line(x1, y1, x2, y2);

Remember: In the arguments to line (and all other drawing functions) the X coordinate comes before the Y coordinate.

Your complete code will probably look something like this: 
line(x1, y1, x2, y2);
