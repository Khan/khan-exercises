(1) What is a vertical asymptote? A vertical asymptote is an x coordinate on the graph where (a) the function is undefined and (b) the function evaluates to larger and larger y values (in either the positive or negative direction) as the x values get closer and closer to this x coordinate.

For example, say we have some function g(x), and it has a vertical asymptote at x = 9. What would that mean? First, it would mean that if we plugged in x = 9 (i.e. we evaluated g(9)), we would get an undefined y value. Second, it would mean that if we plugged in x values closer and closer to 9, our y values would get larger and larger. So g(8) might equal 10, g(8.5) might equal 150, g(8.9) might equal 995, g(8.99) might equal 20,000, and so on.

(2) How do we go about finding vertical asymptotes? Let's start by seeing when f(x) is undefined. A rational function such as f(x) is undefined when the denominator is equal to zero. How can we determine which x values make the denominator evaluate to zero? Let's try factoring. Doing so, we find that the denominator can be rewritten as (x + ASYM1)(x + ASYM2). So if x = NEG_ASYM1 or if x = NEG_ASYM2, our denominator equals zero and our function is undefined.

Now we need to check whether the y values approach positive or negative infinity as the x values approach NEG_ASYM1 and NEG_ASYM2.
