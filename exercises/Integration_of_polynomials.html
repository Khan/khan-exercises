An equation of the form (10/3) x^(3/2)+4x+C Coeffiecients must be integers or fractions (no rounded decimal coefficients).

Recall that the antiderivative of a function of the form : \color{blue} {f(x)=Cx^n} is \color{green} {\dfrac {C}{n+1}x^{n+1}}.

For example, the antiderivative of the function \color{blue} {f(x)=\dfrac {5}{3} x^4 + 3x } is \color{green} {\dfrac {5}{3(4+1)}x^{4+1} + \dfrac{3}{1+1}x^{1+1} + C }, where C is an arbitrary constant or \color{green} {\dfrac {1}{3}x^5 +\dfrac{3}{2}x^2 + C } after simplifying.

When taking the antiderivative of polynomial functions, you add \color{purple}{one} to the exponent \color{purple} {(n+1)} and divide the coefficient by \color{purple}{(n+1)}. You will do this to each term in the function that is of the form \color{purple}{f(x)=Cx^n}.

For this exercise we are evaluating \textbg{indefinite } integrals. Recall that the derivative of a constant term is \color{red} 0, so we must add a constant term to the solution.

Example: \color{blue} { {\displaystyle{\int{3x^4}\,{\mathrm{d}x}} }\,=\,\color{green} { \dfrac{3}{5}x^5\,+\,Constant} }
