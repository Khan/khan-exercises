A cat is stuck in a tree HEIGHT feet tall and surrounded by bushes for WIDTH feet around the tree. How long does a ladder need to be for a fireman to reach the cat while being outside the bushes?

The distances between the fireman and the tree base, the cat and the tree base, and the fireman and the cat form a triangle. Using Pythagorean's theorem, we know that a^2 + b^2 = c^2.

We want to find the length of the ladder, c, with depth of the bushes, a = WIDTH and height of the tree, b = HEIGHT.

c^2 = WIDTH^2 + HEIGHT^2 = LAD_LEN2.

c = \sqrt{LAD_LEN2}.

Simplifying the radical gives c = formattedSquareRootOf(LAD_LEN2).
