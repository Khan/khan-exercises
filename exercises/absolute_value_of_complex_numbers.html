Determine the absolute value of the following complex number:

|REAL + IMAGi|

ABS_SQUARE

The absolute value of any number is its distance from zero. Absolute values of complex numbers can be determined using the Pythagorean theorem.

graphInit({ range: [[-10, 10], [-10, 10]], scale: 20, tickStep: 1, labelStep: 1, }); label( [10, 0.5], "Re", "left"); label( [0.5, 9], "Im", "right"); circle( [REAL, IMAG], 3 / 20, { fill: "blue", stroke: "none" }); label( [REAL, IMAG], REAL + " + " + IMAG + "i", "left", { color: "blue", labelDistance: 4 } ); path([ [0,0], [REAL, 0], [REAL, IMAG]], { stroke: "blue" }); path([ [0,0], [REAL, IMAG]], { stroke: "red" });

The distance of REAL + IMAGi is \sqrt{(REAL)^2 + (IMAG)^2}.

\sqrt{(REAL)^2 + (IMAG)^2} = \sqrt{ABS_SQUARE}

Simplifying the radical gives formattedSquareRootOf(ABS_SQUARE).
