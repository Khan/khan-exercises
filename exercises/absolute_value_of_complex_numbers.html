Determine the absolute value of the following complex number:

REAL + IMAGi

ABS_SQUARE

The absolute value of any number is its distance from zero. As complex numbers can be visualized as points on the complex plane, absolute values of complex numbers can be determined using the distance formula.

graphInit({ range: [[-10, 10], [-10, 10]], scale: 20, tickStep: 1, labelStep: 1, }); label( [10, 0.5], "Re", "left"); label( [0.5, 9], "Im", "right"); circle( [REAL, IMAG], 3 / 20, { fill: "blue", stroke: "none" }); label( [REAL, IMAG], REAL + " + " + IMAG + "i", "left", { color: "blue", labelDistance: 4 } ); path([ [0,0], [REAL, 0], [REAL, IMAG]], { stroke: "blue" }); path([ [0,0], [REAL, IMAG]], { stroke: "red" });

The triangle shown above is a right one. Its two legs are abs(REAL) and abs(IMAG). Their lengths correspond to the absolute values of the real and imaginary parts of the complex number REAL + IMAGi. The length of the hypotenuse is the distance of REAL + IMAGi from zero, or the absolute value we want to find.

Substituting into the Pythagorean theorem, |REAL + IMAGi|^2 = abs(REAL)^2 + abs(IMAG)^2, so |REAL + IMAGi| = \sqrt{abs(REAL)^2 + abs(IMAG)^2}.

\sqrt{abs(REAL)^2 + abs(IMAG)^2} = \sqrt{REAL * REAL + IMAG * IMAG} = \sqrt{ABS_SQUARE}

Simplifying the radical gives formattedSquareRootOf(ABS_SQUARE). That is the absolute value of |REAL + IMAGi|.

The radical cannot be simplified. The absolute value of |REAL + IMAGi| is \sqrt{ABS_SQUARE}.
