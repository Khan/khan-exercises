First, we need to find a common denominator. The least common multiple of \blue{D1} and \pink{D2} is the smallest possible common denominator.

+

We find that \lcm(\blue{D1}, \pink{D2}) = \green{LCM}

+

Now, we need to change both fractions to have a denominator of \green{LCM}. In order to do this, we can multiply \blue{fraction( N1, D1 )} by fraction( F1, F1 ) and \pink{fraction( N2, D2 )} by fraction( F2, F2 ) . These are equal to 1 , so they will change the denominator without changing the value of the fraction.

+

\begin{align*}\blue{fraction( N1, D1 )}\cdot fraction( F1, F1 ) &= \green{fraction( N1 * F1, LCM )}\\ + \pink{fraction( N2, D2 )}\cdot fraction( F2, F2 ) &= \green{fraction( N2 * F2, LCM )}\end{align*}
