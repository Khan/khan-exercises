When thinking about abstract logical statements it often helps to think about what is being said in plain English:

  • If S1 then S2
  • not S2
  • Therefore…?

It also helps to replace the logical variables with example statements, for example:

  • S1= the dog smells a squirrel
  • S2= the dog will bark

So if you substitute the example statements for the logical variables you get:

  1. If the dog smells a squirrel, then the dog will bark.
  2. The dog did not bark
  3. Therefore…?

What can you say is true if the dog did not bark and if the dog smells a squirrel, the dog will bark??

Answer: the dog did not smell a squirrel, or not S1
