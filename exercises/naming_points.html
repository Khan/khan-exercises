What ordered pair defines the point on the graph below?


graphInit({ range: 11, scale: 20, axisArrows: "<->", tickStep: 1, labelStep: 1, gridOpacity: 0.05, axisOpacity: 0.2, tickOpacity: 0.4, labelOpacity: 0.5 }); label([0, 11], "y", "above"); label([11, 0], "x", "right"); graph.point = circle([X, Y], 0.2, { stroke: ORANGE, fill: ORANGE });

( X , Y )

We can find the ordered pair of a point by finding the \blue{x} and \pink{y} value that corresponds to the point.

The \blue{x} value, which goes first, is found by looking first at the horizontal x-axis and finding the value on that axis that is lined up vertically with the point. For this point that is \blue{x = X}.

style({ stroke: BLUE, strokeWidth: 1.5, }, function() { line([X, -11], [X, 11]); }); graph.point.toFront();

The \pink{y} value, which always goes second in an ordered pair, is found by looking at the vertical y-axis and finding the value on that axis that is lined up horizontally with the point. For this point, that value is \pink{y = Y}.

style({ stroke: PINK, strokeWidth: 1.5, }, function() { line([-11, Y], [11, Y]); }); graph.point.toFront();

Since we found that \blue{x = X} and \pink{y = Y} for this point, the ordered pair is (X, Y).
