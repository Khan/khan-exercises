The equation is given to you in the form \color{BLUE}{M(x,y)} + \color{GREEN}{N(x,y)}y'=0.

To determine if this is an exact equation compare M_{y}=\frac{\partial}{\partial y}M(x,y) and N_{x}=\frac{\partial}{\partial y}N(x,y). If \color{BLUE}{M_{y}} = \color{GREEN}{N_{x}}, the equation is exact. [ Show me ]

\color{PURPLE}{\psi_{xy}=} M_y = \frac{\partial}{\partial y}\Big( M.tex() \Big) =M_y.tex()

\color{PURPLE}{\psi_{yx}=} N_x = \frac{\partial}{\partial x}\Big( N.tex() \Big)=N_x.tex()

Since \color{BLUE}{M_y} = \color{PURPLE}{\psi_{xy}} = \color{GREEN}{N_x}, the equation is exact.

To solve for \psi, start by either [ Integrating M ] or [ Integrating N ]

\color{PURPLE}{\psi} = \color{BLUE}{\int M\mathrm{d} x} + f(y) = psi_f.tex()+f(y)

\color{PURPLE}{\psi} = \color{GREEN}{\int N\mathrm{d} y} + g(x) = psi_g.tex()+g(x)

Now finish by find the remaining function      [ Find f(y) ]      [ Find g(x) ]

To find f(y), note that

\qquad \begin{align} f(y)& =\color{PURPLE}{\psi}-\color{BLUE}{\int M \mathrm{d} x} \\ f'(y)& =\frac{\partial}{\partial y}\Big(\color{PURPLE}{\psi} - \color{BLUE}{\int M \mathrm{d} x} \Big) & \text{by differentiation} \\ & = \color{GREEN}{N} - \color{BLUE}{\int M_{y}\mathrm{d} x} & \text{because } \color{PURPLE}{\psi_{y}}=\color{GREEN}{N} \end{align}
