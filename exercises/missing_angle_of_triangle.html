In the triangle below, what is the value of angle labeled X?

init({ range: [[-1, 15 ], [ -8, 1 ]] }) var tr = new Triangle([ 2, -5 ], TRIANGLE, 9, { "points": ['A', 'B', 'C'], "angles" : [TRIANGLE[0] + "^{\\circ}", TRIANGLE[1] + "^{\\circ}", "X"] }); tr.rotate(randRange(0, 90)); tr.boxOut([[[-10, 1], [-10, 1]]], [0, -0.5]) tr.draw(); tr.drawLabels();

X = TRIANGLE[2]\Large{^\circ}

Remember that the three angles in any triangle sum to 180{^\circ}.

We know two angles of the triangle, so the third angle is 180{^\circ} minus the other two.

X = 180{^\circ} - TRIANGLE[0]{^\circ} - TRIANGLE[1]{^\circ} = TRIANGLE[2]{^\circ}
