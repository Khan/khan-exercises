What is the rank of A.toCode('[')?

The rank of a matrix is the number of pivots in the reduced row echelon form.

A.toCode('[')\longrightarrowA_RREF.asSymbolic().map(decimalToFraction).toCode('[')

The pivot columns are RREF_STEPS['pivot_col'][0]['col']+1, PIVOT_POS['col']+1.

The number of pivots is PIVOTS, so the rank of the matrix is PIVOTS.
