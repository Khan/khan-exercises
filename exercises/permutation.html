How many permutations are there of N distinct objects?

PERMUTATIONS

The number of permutations of N distinct objects is N factorial.

If you have three letters, XYZ, they can be arranged as XYZ, XZY, YXZ, YZX, ZXY, or ZYX.

Thus, the number of permutations of N distinct objects is PERMUTATIONS.

N! = PERMUTATIONS
