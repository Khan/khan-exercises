A matrix A is singular if and only if it does not have a well-defined inverse A^{-1}.
To determine whether a matrix is singular, calculate its determinant, which will be zero if the matrix is singular.

The determinant of a 2x2 matrix is given by the following formula:

\begin{vmatrix}a & b\\ c & d\end{vmatrix}= a * d - b * c

Using the formula, we get:

\begin{vmatrix}A11 & A12\\A21 & A22\end{vmatrix} = A11 * A22 - A21 * A12 = DET

The determinant is zero, so the matrix is singular.

The determinant is nonzero, so the matrix is not singular.
