Suppose f is continuous, f(0) = 0, f(B) = B and f'(x) > 0 for x \in [0, B]. Further, suppose that \int_0^{B} f(x)dx = INT. Find the value of \int_0^{B} f^{-1}(x)dx.

ANS

Try drawing an example graph for some f meeting the criteria above. How are the graphs of f and f^{-1} related?

graphInit({ range: [[0,1],[0,1]], scale: 300 }); style({ stroke: GREEN, fill: "none", arrows: null }, function() { plot(function(x) { return Math.pow(x,3); }, [0,1] ); label([0.3, 0.1],"f", {color: GREEN}); label([1,-0.05],B); label([-0.05,1],B); } );

The graph of f^{-1} is found by reflecting the graph of f about the line y = x.

style({ stroke: BLUE, fill: "none", arrows: null }, function() { plot(function(x) { return Math.pow(x,1/3); }, [0,1] ); label([0.63, 0.9],"f^{-1}", {color: BLUE}); } );

The integral \int_0^{B} f(x)dx = INT represents the area of the region R under f.

var fillPoints = []; fillPoints[0] = [0,0]; for (var i = 1; i < 500; i++) { fillPoints[i] = [i/500, Math.pow(i/500,3)]; } fillPoints[500] = [1,1]; fillPoints[501] = [1,0]; style({ stroke: "none", fill: "#eeddaa", opacity: FILL_OPACITY }, function() { path(fillPoints); label([0.75,0.22], "R", {color: "#550000"}); label([0.75,0.12], "Area =" + INT, {color: "#550000"}); });

Notice that the region above f^{-1} and below y= B is the region R reflected about y=x and so also has area equal to INT.

var fillPoints = []; fillPoints[0] = [0,0]; for (var i = 1; i < 500; i++) { fillPoints[i] = [i/500, Math.pow(i/500,1/3)]; } fillPoints[500] = [1,1]; fillPoints[501] = [0,1]; style({ stroke: "none", fill: "#ffddaa", opacity: FILL_OPACITY }, function() { path(fillPoints); label([0.3,0.88], "Area =" + INT, {color: "#550000"}); });

The integral of f^{-1} over [0,B] represents the area of the region below f^{-1}.

var fillPoints = []; fillPoints[0] = [0,0]; for (var i = 1; i < 500; i++) { fillPoints[i] = [i/500, Math.pow(i/500,1/3)]; } fillPoints[500] = [1,1]; fillPoints[501] = [1,0]; style({ stroke: "none", fill: BLUE, opacity: FILL_OPACITY }, function() { path(fillPoints); });

So, \int_0^{B} f^{-1}(x)dx = (area of B \times B square) - (Area of region R) = SQ - INT = ANS.

Another way to approach this problem is to notice that the graph of x = f^{-1}(y) (i.e. the graph of f^{-1} when plotted as a function of y) coencides with the graph of y = f(x). So, \int_0^{B} f^{-1}(y)dy represents the area between y = f(x) and the y-axis and we once again have \int_0^{B} f^{-1}(y)dy = (area of B \times B square) - (Area of region R).
