Solve for x:  BASE^{\log x} = RIGHTSIDE

ANSWER

If we look carefully, the right side is a power of BASE:  BASE^{\log x} = BASE^{POWER}

If the bases are equal, then the powers must be equal!

If this isn't too clear, we can always take the \log of both sides and simplify:

\log BASE^{\log x} = \log BASE^{POWER}

\log x \cdot \log BASE = POWER \cdot \log BASE

\log x = POWER

10^{POWER} = x

Therefore x = ANSWER.
