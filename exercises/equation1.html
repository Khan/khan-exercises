In this equation you can see similar fractions which should lead you to the idea of finding their \underline {\text{partial fraction decomposition}}

Use the following : \dfrac{1}{X}-\dfrac{1}{X+A}=\dfrac{(X+A)-X}{X(X+A)}=\dfrac{A}{X(X+A)} for any number A

In this case : \left(\dfrac{1}{X}-\dfrac{1}{X+1}\right)= \dfrac{1}{X}-A - \left(\dfrac{1}{X+1} - \dfrac{1}{X-M}\right)

It follows that : 0 = -A + \dfrac{1}{X-M}

Then : X-M = \dfrac{1}{A}
