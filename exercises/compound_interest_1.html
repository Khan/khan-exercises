FV

Since the interest compounds compounding, we receive 1/n of rate% interest at the end of each period.

Adding in 1/n of rate% interest is the same as multiplying by (expr( [ "+", 1, [ "frac", r, n ] ] )).

Since we are adding in this interest n times each year for years, we will multiply by (expr( [ "+", 1, [ "frac", r, n ] ] )) a total of n*T = n * T times.

So the amount in the account at the end of years is expr( [ "*", PV, [ "^", [ "+", 1, [ "frac", r, n ] ], [ "*", n, T ] ] ] ), or about $FV.
