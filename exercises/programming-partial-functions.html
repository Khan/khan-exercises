Creating a partial function means that you take a function, like "add_three_numbers(x,y,z)", and fill in one of the arguments with a constant number. So making a partial function out of the function from before would mean we would have a function that adds two numbers to some fixed constant number. Write a function called "create_partial_function" that takes in a function and an argument and returns a partial function. Assume that the function to be partial only takes 3 arguments.
  • example_func = lambda x,y,z: (x+y)/z
  • create_partial_function(example_func,1)(2,3)==1
