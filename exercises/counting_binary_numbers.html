Represent the following Base 10 (Decimal) number in Base 2 (Binary).

What is INTEGER in Binary?

SOLUTION

Remember, in Base 2 we have a 'twos' place instead of a 'tens' place.

If the number provided is larger than 2, you'll have to add places.

Base 2 is often represented with four places, example: 0000. From right to left the places are: ones, twos, fours, eights.

How many of each place do you have? Try adding the place values up.

If you are stuck, try watching the Binary Numbers lesson again.
