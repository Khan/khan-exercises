Simplify the following expression:

\dfrac{X^2 + CONSTANT}{X + A}

an expression, like x+1

The numerator is in the form \color{PINK}{a^2} - \color{BLUE}{b^2}, which is a difference of two squares so we can factor it as (\color{PINK}{a} + \color{BLUE}{b}) (\color{PINK}{a} - \color{BLUE}{b}).

\qquad a = X

\qquad b = \sqrt{A * A} = A

So we can rewrite the expression as: \dfrac{(\color{BLUE}{X} + \color{PINK}{A}) (\color{BLUE}{X} \color{PINK}{-A})} {X + A}

So we can rewrite the expression as: \dfrac{(\color{BLUE}{X} \color{PINK}{A}) (\color{BLUE}{X} + \color{PINK}{-A})} {X + A}

We can now divide the numerator and denominator by (X + A).

Which leaves us with X + -A.
