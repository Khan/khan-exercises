Solve using the properties of logarithms.

\log_{BASE}((\frac{A}{B})^POWER) = {?}

(X1 - X2)*POWER

First use the rule: n \cdot \log(a) = \log(a^{n}).

\log_{BASE}((\frac{A}{B})^POWER) = POWER\cdot \log_{BASE}(\frac{A}{B})

Next use the rule: \log(a) - \log(b) = \log(\frac{a}{b}).

POWER\cdot \log_{BASE}(\frac{A}{B}) = POWER\cdot (\log_{BASE}(A) - \log_{BASE}(B))

= POWER\cdot (X1 - X2)

= POWER*(X1 - X2)

Solve using the properties of logarithms.

\log_{BASE}((A \cdot B)^POWER) = {?}

(X1 + X2)*POWER

First use the rule: n \cdot \log(a) = \log(a^{n}).

\log_{BASE}(({A \cdot B})^POWER) = POWER\cdot \log_{BASE}(\frac{A}{B})

POWER\cdot \log_{BASE}(A \cdot B) = POWER\cdot (\log_{BASE}(A) - \log_{BASE}(B))

= POWER\cdot (X1 + X2)
