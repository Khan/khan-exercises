randRange(1,9) N/10

Express N/10 as a decimal.

N/10

We have N over 10, so there are N tenths.

If we have N tenths, we can write them as N/10

randRange(1,99) N/100

Express N/100 as a decimal.

N/100

We have N over 100, so there are floor(N / 10) tenths.

There are floor(N % 10) hundredths.

If we have floor(N / 10) tenths, we can write them as (N - floor(N % 10))/100.

If we have floor(N % 10) hundredths, we can write them as floor(N % 10)/100.

If we have floor(N / 10) tenths and floor(N % 10) hundredths, we can write them as N/100.
