Express fraction( N, D ) as a decimal.

DEC

The decimal form is just the numerator with the decimal place moved left by the number of 0s in the denominator.

For division by D, this means D === 10 ? "1 place" : "2 places".

If you move the correct number of decimal places, you get DEC.

randRange( 0, 9) randRange( 0, 9 ) randRange( 0, 9 ) (TEN/10) + (HUND/100) + (THOU/1000)

Write the following as a decimal:

TEN tenths, .HUND hundredths, .THOU thousandths.

DEC

The first place to the right of the decimal is the tenths place.

The word next to the number tells you want to divide that number by. So the tenths would be fraction( TEN, 10 ), or simply (TEN/10).toFixed(1).

Add the different values together: fraction(TEN, 10) + fraction(HUND, 100) + fraction(THOU, 1000) or (TEN/10).toFixed(1) + (HUND/100).toFixed(2) + (THOU/1000).toFixed(3)

The final answer is: DEC.
