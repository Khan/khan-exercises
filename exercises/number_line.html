init({ range: [ [ -6, 6 ], [ -1, 1 ] ] }); line( [ -5, 0 ], [ 5, 0 ] ); for ( var x = -5; x <= 5; x++ ) { line( [ x, -0.2 ], [ x, 0.2 ] ); } style({ stroke: "#6495ED", strokeWidth: 3.5 }); line( [ 0, -0.2], [0, 0.2]); label( [ 0, -0.53 ], "0", "center", { color: "#6495ED" }); style({ stroke: "#FFA500", fill: "#FFA500" }); graph.orangeDot = circle( [ NUMBER, 0 ], 0.10 );
NUMBER

We know where 0 is on this number line because it is labeled.

Numbers to the right of 0 are positive, while numbers to the left of 0 are negative.

Starting from 0, we move abs( NUMBER ) to the leftright to reach the orange dot.

style({ stroke: "#6495ED", fill: "#6495ED", strokeWidth: 3.5, arrows: "->" }); line( [ 0, 0 ], [ NUMBER, 0 ] ); graph.orangeDot.toFront();

Thus, the orange dot represents the number NUMBER.
