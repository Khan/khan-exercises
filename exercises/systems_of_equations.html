One way to solve a system of equations (but not the only way) is to eliminate either the x variable or the y variable.

We do this by making the x (or y) coefficients be the same number, which we do by multiplying top and bottom equations based on Least Common Multiple principles.

For the x coefficients, we need to multiply top and bottom by MULT_1 and MULT_2.

For the y coefficients, we need to multiply top and bottom by MULT_3 and MULT_4.

Thus, it is easiest to line up the (XY_FLAG ? "x" : "y") coefficients (the (XY_FLAG ? "x" : "y") multipliers are generally smaller).
