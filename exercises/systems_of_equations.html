NOTE: There are lots of ways to solve this, watch the videos for more info. Below is one approach that works, but not the only one.

\color{red}{expr(["*", A1, "x"])}{} + B1y = C1
\color{red}{expr(["*", A2, "x"])}{} + B2y = C2

You want either 1) both x coefficients to be the same or 2) both y coefficients to be the same so you can add or subtract one equation from the other.

Ideally, you want to do it with the least amount of complicated math, so which method, 1) or 2) is the easiest to do?

For the x coefficients, we need to multiply top and bottom by MULT_1 and MULT_2 (to get to the Least Common Multiple of A1 and A2).

For the y coefficients, we need to multiply top and bottom by MULT_3 and MULT_4 (to get to the Least Common Multiple of B1 and B2).

Thus, it is easiest to line up the (XY_FLAG ? "x" : "y") coefficients (the (XY_FLAG ? "x" : "y") multipliers are generally smaller).

You can muliply the top equation by MULT_1 and the bottom equation by MULT_2 to get:

expr(["+", ["*", A1*MULT_1, "x"], ["*", B1*MULT_1, "y"]]) = C1*MULT_1
expr(["+", ["*", A2*MULT_2, "x"], ["*", B2*MULT_2, "y"]]) = C2*MULT_2

(This step is an optional step, we are listing it here for clarity and to show you each step in the process)

Adding the top and bottom equations gives you:

expr(["*", B1*MULT_1+B2*MULT_2, "y"]) = C1*MULT_1+C2*MULT_2

Dividing both sides by B1*MULT_1+B2*MULT_2 and then reducing terms gives:

y = fractionReduce( Y_NUMER, Y_DENOM )

Now you can substitute fractionReduce( Y_NUMER, Y_DENOM ) back into the top equation to get:

expr(["+", ["*", A1, "x"], ["*", B1, fractionReduce( Y_NUMER, Y_DENOM )]]) = C1

expr(["+", ["*", A1, "x"], fractionReduce( B1 * Y_NUMER, Y_DENOM )]) = C1

expr(["*", A1, "x"]) = fractionReduce( C1 * Y_DENOM - B1 * Y_NUMER, Y_DENOM )

x = fractionReduce(X_NUMER,X_DENOM)

Therefore, the answer is: x = fractionReduce(X_NUMER,X_DENOM), y = fractionReduce( Y_NUMER, Y_DENOM )
