The function f(x) = \lfloor x \rfloor defined by:

        \lfloor x \rfloor = the greatest integer less than or equal to x.

is called the greatest integer function or the floor function. For example: \lfloor 2.45 \rfloor = 2 and \lfloor \pi \rfloor = 3.

Find \int_{A}^{B} \lfloor x \rfloor dx.

ANS

Try drawing a graph of f(x) = \lfloor x \rfloor to get an idea of what the area under it looks like.

For any value 3 \leq x < 4, we have \lfloor x \rfloor = 3. Therefore, the graph of f(x) = \lfloor x \rfloor between x=3 and x=4 looks like a horizontal line segment at y=3.

graphInit({ range: 6, scale: 30, axisArrows: "<->", tickStep: 1, labelStep: 1, gridOpacity: 0.05, axisOpacity: 0.2, tickOpacity: 0.4, labelOpacity: 0.5 }); label( [ 0, -6 ], "y", "below" ); label( [ 6, 0 ], "x", "right" ); graph.set = raphael.set(); style({ stroke: GREEN, strokeWidth: 2, }, function() { var i = 0; for (i = -6; i < 6; i++) { graph.set.push( line( [ i, i ], [ i+1, i ] ) ); } });

\int_{A}^{B} \lfloor x \rfloor dx is the area under \lfloor x \rfloor between x = A and x=B.

style({ fill: "#aaffaa", stroke: "#88cc88" }, function() { var i = 0; for (i = A; i < B; i++) { graph.set.push( path( [[i, i], [i, 0],[i+1, 0],[i+1, i]] ) ); } });

Since the area is made up of rectangles, each of width 1, we can determine the integral simply by adding up the areas of these boxes. Remember that area below the x-axis counts as negative.

So, \int_{A}^{B} \lfloor x \rfloor dx = ANS.
