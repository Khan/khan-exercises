A line is defined by the following equation:

expr([ "*", X_COEFF, "x" ]) + expr([ "*", Y_COEFF, "y" ]) = EQUALS

What is the value of the y-intercept?

\Large{(}\ 0,Y_INTERCEPT\Large{)}

The y-intercept is the point where the line crosses the y-axis. This happens when x is zero.

Set x to zero and solve for y.

expr([ "*", X_COEFF, 0 ]) + expr([ "*", Y_COEFF, "y" ]) = EQUALS

expr([ "*", Y_COEFF, "y" ]) = EQUALS

(fractionReduce( 1, Y_COEFF )) \cdot (Y_COEFFy) = (fractionReduce( 1, Y_COEFF )) \cdot (EQUALS)

y = Y_INTERCEPT

This line intersects the y-axis at (0, Y_INTERCEPT).
