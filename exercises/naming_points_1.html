What ordered pair defines the point below:

graphInit({ range: 10, scale: 20, axisArrows: "<->", tickStep: 1, labelStep: 1 }) style({ stroke: "#6495ED", fill: "#6495ED" }, function() { circle( POINT, 0.15 ); });

(POINT[0] ,POINT[1] )

We can find the ordered pair of a point by finding the x and y value that corresponds to the point.

The x value, which goes first, is found by looking first at the horizontal x axis and finding the value on that axis that is lined up vertically with the point.

For this point that is x=POINT[0].
line( POINT, [POINT[0], 0], { stroke: "#ffa500", arrows: "->" } ) style({ stroke: "#6495ED", fill: "#6495ED" }, function() { circle( POINT, 0.15 ); });

The y value, which always goes second in an ordered pair, is found by looking at the vertical y axis and finding the value on that axis that is lined up horizontally with the point.

For this point, that value is y=POINT[1].
line( POINT, [0, POINT[1]], { stroke: "#28ae7b", arrows: "->" } ) style({ stroke: "#6495ED", fill: "#6495ED" }, function() { circle( POINT, 0.15 ); });

Since we found that x=POINT[0] and y=POINT[1] for this point, the ordered pair is (POINT[0],POINT[1]).
