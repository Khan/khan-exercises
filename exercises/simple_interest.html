If you deposit $PRINCIPAL.00 into an account which pays RATE% simple interest and you leave it in that account for TIME years.

How much interest will have accrued? (Round to the nearest penny)

Interest = $ SIMPLE_INTEREST

When discussing simple interest,the interest accrued is the same in each time period.

Interest for 1 time period = Principal * Rate

Accrued Interest = Interest Per Period * Number of Periods
= ( $PRINCIPAL * RATE/100 ) * TIME = $ SIMPLE_INTEREST

What will be the balance after TIME years? (Round to the nearest penny)

Balance = $ PRINCIPAL + SIMPLE_INTEREST

To find the final balance, add the interest accrued to the initial balance

Interest per period = Initial Balance * Interest Rate
= PRINCIPAL * RATE/100
= Math.roundTo(2,PRINCIPAL * RATE/100)

Interest Accrued = Interest Per Period * Number of Time periods
= roundTo(2,PRINCIPAL * RATE/100) * TIME
= SIMPLE_INTEREST

Final Balance = Initial Balance + Interest Accrued
= PRINCIPAL + SIMPLE_INTEREST
= PRINCIPAL + SIMPLE_INTEREST
